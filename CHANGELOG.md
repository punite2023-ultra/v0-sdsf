# Changelog

All notable changes to the SDSF website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-07

### Added
- Initial release of SDSF - Social Digital Selling Force website
- 8 complete pages: Home, About, Services, Portfolio, Pricing, Blog, Careers, Contact
- 5 in-depth blog articles on TikTok live selling and e-commerce strategies
- 6 detailed job openings with full requirements and benefits
- Premium color scheme (gold accents with black and white)
- Dark mode support with CSS variables
- Scroll-triggered animations on all sections
- Mobile-responsive design (works on all devices)
- Contact form with validation
- Blog listing with individual article pages
- Expandable job accordion on careers page
- Navigation with mobile menu toggle
- SEO optimization with metadata on all pages
- GitHub Actions CI/CD pipeline
- Vercel deployment ready
- Comprehensive documentation

### Tech Stack
- Next.js 16 with App Router
- React 19.2
- TypeScript 5.7
- Tailwind CSS 4
- shadcn/ui (50+ components)
- Lucide React icons
- React Hook Form for validation
- Zod for schema validation

### Content
- Blog Articles:
  1. How to prepare for TikTok live selling
  2. Why ads fail without creators
  3. How TikTok Shop brands fail in first 60 days
  4. Live selling script framework
  5. Shopee vs TikTok conversion strategy

- Job Openings:
  1. TikTok Live Selling Host ($1,500-$2,500/month)
  2. E-Commerce Content Creator ($1,200-$2,000/month)
  3. E-Commerce Operations Manager ($1,800-$2,800/month)
  4. Social Media & Community Manager ($1,000-$1,800/month)
  5. Product Photographer & Stylist ($1,200-$2,000/month)
  6. Data Analyst ($1,500-$2,500/month)

### Documentation
- Comprehensive README with quick start guide
- CONTRIBUTING.md with code guidelines
- EDITABLE_CONTENT.md with content editing instructions
- QUICK_START.md for first-time setup
- Multiple customization guides
